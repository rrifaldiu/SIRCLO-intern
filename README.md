
## Problem 3 : Bola Berantai

This problem is solved using C++

To compile Bola Berantai program, using console, type :
```
cd bolaberantai
g++ bolaberantai.cpp -o bolaberantai
```
To run it on windows, type :
```
bolaberantai.exe
```
On Linux and/or MacOS, type :
```
./bolaberantai
```
